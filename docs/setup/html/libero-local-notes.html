Old blockers

What the old blocked record warned about

1. benchmark hard-imports torch

Even task registry or BDDL lookup requested the training stack. The current route reads BDDL directly from the package directory.

2. OffScreenRenderEnv opens GL

The headless M1 had no GL context. The current route uses lower-level ControlEnv without renderers.

3. version drift

The local robosuite 1.5.2 / Python 3.11 stack did not represent production. The current local stack is isolated as Py3.10 + robosuite 1.4.0.
