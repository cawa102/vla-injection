Old blockers

古いblocked記録は、何を警告していたか

1. benchmark hard-imports torch

task registryやBDDL locationを見るだけでもtraining stackを要求していた。 現在はpackage dirから直接BDDLを読む。

2. OffScreenRenderEnv opens GL

headless M1でGL contextがない。現在はlower-level ControlEnv をrendererなしで使う。

3. version drift

local boxのrobosuite 1.5.2 / Python 3.11はproductionを代表しない。 現在はPy3.10 + robosuite 1.4.0に分離。

honest historyとして残す
